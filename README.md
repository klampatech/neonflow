# NEONFLOW

> A visually striking, single-page task manager that feels like a living, breathing digital organism.

**NEONFLOW** is an acid rave meets brutalist productivity — deep void backgrounds shattered by electric neons, chunky brutalist typography, and motion that feels physics-driven and satisfying.

## Features

- 🎨 **Bold Visual Design** — Electric neons, glassmorphism, ambient glow effects
- ⚡ **Zero Dependencies** — Vanilla JS, single HTML file
- 📱 **Responsive** — Mobile-first, works offline
- ⌨️ **Keyboard Navigation** — Full shortcut system for power users
- 🔄 **Drag & Drop** — Reorder tasks, move between lists
- 🔍 **Smart Filters** — Filter by priority, due date, tags
- 💾 **Local Persistence** — Data survives page refreshes

## Quick Start

```bash
# Clone the repo
git clone https://github.com/klampatech/neonflow.git
cd neonflow

# Open in browser
open index.html
```

That's it. No build step, no dependencies, just open `index.html`.

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `⌘/Ctrl + N` | New task |
| `⌘/Ctrl + F` | Search |
| `J/K` or `↑/↓` | Navigate tasks |
| `Space` | Toggle completion |
| `Enter` | Edit task |
| `D` | Delete task |
| `?` | Show shortcuts |

## Tech Stack

- **HTML5** — Single file
- **CSS3** — Custom properties, Grid, Flexbox
- **Vanilla JavaScript** — Zero external dependencies
- **Google Fonts** — Space Grotesk, JetBrains Mono
